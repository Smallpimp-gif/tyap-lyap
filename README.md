# ТЯП-ЛЯП — сайт студии дизайнерского ремонта

Лендинг для студии **ТЯП-ЛЯП** (дизайнерский ремонт под ключ, Москва и область).
Статический сайт без сборки и зависимостей — чистые HTML, CSS и JavaScript.
Готов к размещению на домене **tyap-lyap.ru**.

Концепция: редакторский минимализм в духе Never Too Small / современных дизайн-студий.
Тёплая нейтральная палитра (штукатурка, бетон, графит) + один акцент — терракота.
Фирменный приём — «мазок» (акцентная плашка), он же дефис в логотипе ТЯП‑ЛЯП.

## Структура

```
.
├── index.html              # вся разметка и тексты (одностраничник)
```

подход → материалы → процесс + форматы → о студии → контакты (форма) → подвал.

## Как открыть локально

Нужен любой простой веб-сервер (из-за относительных путей; просто двойной клик по
`index.html` тоже сработает, но сервер надёжнее):

```bash
# из папки проекта
python3 -m http.server 5577
# затем открыть http://localhost:5577
```

## Как опубликовать на tyap-lyap.ru

Это статический сайт — подойдёт любой хостинг:

- **Обычный хостинг / VPS:** загрузите все файлы в корень сайта (`public_html`).
- **Netlify / Vercel / Cloudflare Pages:** перетащите папку проекта — деплой за минуту,
  затем привяжите домен tyap-lyap.ru.
- **GitHub Pages:** запушьте репозиторий и включите Pages для ветки.

HTTPS и сжатие (gzip/brotli) включите на стороне хостинга.

### Форма заявки
Форма сейчас работает в демо-режиме (показывает сообщение «Спасибо», но **никуда не
отправляет данные**). Чтобы получать заявки, подключите бэкенд — самый простой путь
без программирования — [Formspree](https://formspree.io) или [Getform](https://getform.io):

1. Зарегистрируйтесь, получите endpoint-ссылку.
2. В `index.html` у `<form id="leadForm">` добавьте `action="ВАША_ССЫЛКА" method="POST"`.
3. В `assets/js/main.js` в обработчике формы можно убрать `e.preventDefault()` или
   заменить демо-логику на реальную отправку (`fetch`). Комментарий в коде помечен
   «Demo behaviour».

## 🖼 Изображения — важно

Сейчас в `assets/img/` стоят **бесплатные стоковые фото интерьеров с Unsplash**
(лицензия Unsplash, можно использовать коммерчески). Это временные иллюстрации,
подобранные под стиль бренда.

**Проекты в разделе «Избранные объекты» — демонстрационные** (названия, площади
и локации придуманы под верстку). Перед публичным запуском замените их на реальные
фотографии ваших объектов, чтобы не вводить клиентов в заблуждение.

Чтобы заменить фото — просто положите свой файл с тем же именем в `assets/img/`
(или поменяйте путь в `index.html`). Рекомендуемые размеры: ширина 1600 px, JPEG,
качество ~80%. Имена файлов говорят сами за себя: `hero.jpg`, `project-01…06.jpg`,
`cat-kitchen/living/bedroom/bath.jpg` (блок «Что мы делаем»), `about.jpg`,
`band-manifesto.jpg`, `band-cta.jpg`, `detail-*.jpg`.

Фото основателя в блоке «О студии» (`about.jpg`) — стоковое, замените на реальное
фото Никиты.

## Шрифты

Подключены через Google Fonts (нужен интернет у посетителя):
**Manrope** (основной), **IBM Plex Mono** (подписи/метки), **Playfair Display**
(курсивные акценты). Все с поддержкой кириллицы. Заданы системные запасные шрифты,
так что сайт читаем даже без загрузки шрифтов. При желании их можно «вшить» локально
(self-host) для полной автономности.
